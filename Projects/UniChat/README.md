# UniChat 项目介绍

## 项目概述

基于链上关系和身份的聊天室。

## 技术架构

基于 web 端设计的 web3 即时聊天程序，包括前端、后端和合约。

## 合约语言

Solildiy

## 前端框架

Vue

## 核心功能

1. 使用 web3 钱包签名登陆

2. 自动搜索登陆地址代币并拉群

3. 使用地址搜索并加入代币群

4. 显示群成员代币余额
