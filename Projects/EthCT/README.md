# EthCT项目介绍

### 项目概述
基于以太坊实现的 RFC 6962 Certificate Transparency，用来替代中心化的CA(Certificate Authority)，为证书系统的存储，验证提供去中心化的解决方案。

### 技术架构
合约语言：Solidity

### 核心功能

1. Generate Certificate proof on the Ethereum blockchain
2. check current valid proofs
